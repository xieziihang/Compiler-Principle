## 测试代码

```
void main()
{
  int i;
  print i;
  println;
  i++;
  print i;
  println;
  i--;
  print i;
  println;
  int j;
  j = 3;
  print j;
  ++j;
  println;
  print j;
  println;
  --j;
  print j;
}
```
