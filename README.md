## 1. 项目说明

- 项目是基于老师提供的 MicroC 代码：
  - 增加了新的多行注释方式 `(**)`
  - 增加了 `int` 类型的变量自增自减
  - 增加了三目运算符 `... ? ... : ...`
  - 增加了 `for` 循环和 `do ... while` 循环
  - 增加 `switch ... case` 功能
  - 增加 `+=、-=、*=、/=` 功能
  - 增加 `Max(a, b)` 取出 `a`、`b` 中较大者和 `Min(a, b)` 取出 `a` 、`b` 中较小者
  - 增加 `Abs(x)` 取 `x` 绝对值
